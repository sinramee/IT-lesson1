What is SQL?<\/h2>
  • SQL stands for Structured Query Language<\/li>
  • SQL lets you access and manipulate databases<\/li>
  • SQL became a standard of the American National Standards Institute (ANSI) in 1986, and of the International Organization for Standardization (ISO) in 1987<\/li><\/ul><\/p>\n\n\n\n

    What Can SQL do?<\/h2>\n\n\n\n
      \n
    • SQL can execute queries against a database<\/li>\n\n\n\n
    • SQL can retrieve data from a database<\/li>\n\n\n\n
    • SQL can insert records in a database<\/li>\n\n\n\n
    • SQL can update records in a database<\/li>\n\n\n\n
    • SQL can delete records from a database<\/li>\n\n\n\n
    • SQL can create new databases<\/li>\n\n\n\n
    • SQL can create new tables in a database<\/li>\n\n\n\n
    • SQL can create stored procedures in a database<\/li>\n\n\n\n
    • SQL can create views in a database<\/li>\n\n\n\n
    • SQL can set permissions on tables, procedures, and views<\/li>\n<\/ul>\n\n\n\n
